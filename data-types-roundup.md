#### Explore - Data Type Conversions

```python
x = 1
y = "1"
print( x + int(y))
print( str(x) + y)
```

```python
print( bool( 1))
print( bool( 0))
print( bool( "1"))
print( bool( "0"))
```

```python
print( bool( "abc"))
print( bool( 240))
print( bool( -1))
```

```python
print( int( True))
print( int( False))
```

```python
print( bool( str( False)))
```

```python
x = 1.5
print( x)
print( str( x))
print( int( x))
print( 5 * x)
print( 5 * str(x))
print( 5 * int(x))
```
